# ClaudeSkill-TFMAnalisis-documental-subvenciones
Este repositorio público de GitHub aloja las herramientas gratuitas desarrolladas por Jesús Alberto Salazar Cabrera para su TFM en la UAX. Contiene un Skill de Claude, un Gem de Gemini y un prompt maestro para automatizar el análisis documental Go/No-Go de subvenciones en microorganizaciones.
